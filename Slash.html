Разные сочетания клавиш на все случаи жизни

Стандартные сочетания клавиш Windows:

  • CTRL + C — Скопировать

  • CTRL + V — Вставить

  • CTRL + X — Вырезать

  • CTRL + A — Выделить всё

  • CTRL + Z — Отмена

  • CTRL + W — Закрывает выбранное окно

  • CTRL + N — Создать файл

  • CTRL + O — Открыть файл

  • CTRL + S — Сохранить файл

  • CTRL + P — Печать документа

  • WIN + R — Вызывает окно выполнить

  • WIN + C — Панель управления

  • WIN + M — Свернуть все окна

  • WIN + L — Быстрая смена пользователя

  • WIN + BREAKE — Вызывает утилиту Система



Для работы с файлами:

  • WIN + E — Проводник

  • WIN + F — Поиск файлов

  • CTRL + F — Поиск файлов в папке

  • CTRL + N — Создать файл

  • CTRL + O — Открыть файл

  • CTRL + S — Сохранить файл

  • CTRL + P — Печать документа



Обозначение клавиш [F1—F12]

  • F1 — Вызывает справку Windows

  • F2 — Переименовывает выделенный объект

  • F3 — Открывает окно поиска

  • F4 — Ничего

  • F5 — Обновляет выбранное окно

  • F6 — Перемещает курсор в адресную строку браузера

  • F7 — Проверяет правописание

  • F8 — Режим загрузки

  • F9 — Ничего

  • F10 — Выводит строку меню в открытой программе

  • F11 — Включает и выключает полноэкранный режим

  • F12 — Ничего
